Privacy Policy

Effective Date: November 21, 2025

Welcome to the AxelBase HTML Entity Encoder / Decoder — a completely client-side, privacy-first web utility. We believe privacy should not be a feature; it should be the default.

1. We Collect Nothing

This tool does not collect, store, transmit, or process any personal data — ever.

  • No IP logging — We don’t know you’re here
  • No analytics — No Google Analytics, Plausible, or any tracking
  • No cookies — Not even session cookies
  • No input storage — Your text exists only in your browser’s memory
  • No server — This is a static site hosted on GitHub Pages

2. How It Works (100% In Your Browser)

All encoding and decoding happens exclusively on your device using pure TypeScript. When you type or paste text:

  • It is processed instantly by JavaScript running locally
  • No data is sent over the internet
  • Copy-to-clipboard and examples work via browser APIs only
  • Even Bootstrap CSS/JS is loaded from a trusted CDN — but no tracking occurs

3. Offline & Saveable

You can save this page (Ctrl+S) and use it forever offline. It will work exactly the same — because there is no backend to connect to.

4. Third-Party Services

The only external request is Bootstrap 5 from jsDelivr (a public CDN). It serves static files only — no tracking, no profiling.

5. Children’s Privacy

This tool is safe and appropriate for all ages. No personal information is ever requested or collected.

6. Changes to This Policy

Any updates will appear on this page with a new effective date. Given our zero-data model, changes are extremely unlikely.

7. Contact

Questions? Open an issue on our GitHub repository.

Your data never leaves your device.
That’s not a promise — it’s how the tool is built.
